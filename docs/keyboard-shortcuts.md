# Keyboard Shortcuts

Rewind is designed for keyboard-driven workflows. All shortcuts can be rebound in **Settings > Keybindings**.

## Player Controls

| Key         | Action                       |
| ----------- | ---------------------------- |
| Space / K   | Play / Pause                 |
| Left Arrow  | Seek back 5 seconds          |
| Right Arrow | Seek forward 5 seconds       |
| Up Arrow    | Volume up 5%                 |
| Down Arrow  | Volume down 5%               |
| 0-9         | Jump to 0%-90% of video      |
| , (comma)   | Previous frame (when paused) |
| . (period)  | Next frame (when paused)     |
| <           | Decrease playback rate       |
| >           | Increase playback rate       |
| C           | Toggle captions              |
| F           | Toggle fullscreen            |
| T           | Toggle theater mode          |
| I           | Toggle picture-in-picture    |
| M           | Mute / Unmute                |

## Watch Page

| Key     | Action                             |
| ------- | ---------------------------------- |
| Shift+M | Create marker at current position  |
| Shift+I | Set clip in-point                  |
| Shift+O | Set clip out-point                 |
| Shift+C | Create clip from current selection |

## Cut Page

| Key  | Action        |
| ---- | ------------- |
| I    | Set in-point  |
| O    | Set out-point |
| Home | Go to start   |
| End  | Go to end     |

## Hardware Keys (F14-F24)

For users with extended keyboards or macro pads:

| Key | Action                            |
| --- | --------------------------------- |
| F14 | Set clip in-point                 |
| F15 | Set clip out-point                |
| F16 | Create clip from selection        |
| F17 | Play / Pause                      |
| F18 | Seek back 10 seconds              |
| F19 | Seek forward 10 seconds           |
| F20 | Previous frame                    |
| F21 | Next frame                        |
| F22 | Create marker at current position |

## Media Keys

| Key            | Action                  |
| -------------- | ----------------------- |
| Play / Pause   | Play / Pause            |
| Previous Track | Seek back 10 seconds    |
| Next Track     | Seek forward 10 seconds |

## Customization

Open **Settings > Keybindings** to rebind any action. Changes apply immediately and are stored per user. Click any row to reassign the key, then press the new key combination.
